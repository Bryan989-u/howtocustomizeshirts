Screen Printing

Now, we have the screen-printing method. This is usually used for large productions because it is cost-effective. Trying to do small orders would cost a lot more than what needs to be charged. This method requires a lot of steps, items, and a learning curve. You might not get it perfect the first time, and that is alright. It can take some time to perfect it.

We can start with the equipment:

  • Mesh Screen (110 mesh count recommended)
  • Emulsion
  • Transparent film
  • Exposure Unit. (UV light, exposure box, or sunlight)
  • An area to wash your screen. (The area will get messy, so I donâ€™t suggest doing it inside your house. I would suggest a power washer)
  • Squeegee (based on the size of the screen)
  • Pallet (flat surface)
  • Conveyor dryer, heat press, or heat gun
  • Tape
  • Spatula
  • Shirt (100% cotton is recommended)

These are additional, but it will help if you are running a large production:

  • Screen printing press
  • Flash dryer
  • Degreaser

Now we can get to the screen-printing method:

  1. You can start making the design in an appropriate software
  2. Print it out on a transparent film
  3. Prepare the screen. This is done by coating with the emulsion
  4. You will let it dry in the dark
  5. Next, you will expose the screen with the UV light, exposure box, or sunlight
  6. You will then wash the screen with a power washer
  7. Set up the screen on the shirt where you would like to print the design
  8. Place a small amount of ink across the screen and use the squeegee to spread it out
  9. Lastly, once the design has been printed on a shirt, let the ink dry with either a heat press, conveyor dryer, or heat gun
